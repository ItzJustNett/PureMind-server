🚀 DEPLOYMENT STEPS (Run on Server):
═════════════════════════════════════

Step 1: Install Dependencies
─────────────────────────────
ssh user@api.xoperr.dev

sudo apt update
sudo apt install -y python3 python3-pip nginx certbot python3-certbot-nginx

pip3 install fastapi==0.104.1 uvicorn==0.24.0 python-multipart==0.0.6 \
    httpx==0.25.0 aiofiles==23.2.1 --break-system-packages


Step 2: Get SSL Certificate
───────────────────────────
sudo certbot certonly --nginx -d api.xoperr.dev
# Follow the prompts


Step 3: Setup Nginx
──────────────────
sudo cp lessons-api.service /etc/systemd/system/
sudo cp nginx.conf /etc/nginx/sites-available/api.xoperr.dev
sudo ln -s /etc/nginx/sites-available/api.xoperr.dev /etc/nginx/sites-enabled/
sudo rm -f /etc/nginx/sites-enabled/default

sudo nginx -t
sudo systemctl restart nginx


Step 4: Edit OpenRouter API Key
────────────────────────────────
# Edit the service file to add your actual API key:
sudo nano /etc/systemd/system/lessons-api.service

# Find this line:
Environment="OPENROUTER_API_KEY=sk_your_api_key_here"

# Replace sk_your_api_key_here with your actual key, then save (Ctrl+O, Enter, Ctrl+X)


Step 5: Start the Service
─────────────────────────
sudo systemctl daemon-reload
sudo systemctl enable lessons-api
sudo systemctl start lessons-api

# Verify
sudo systemctl status lessons-api


Step 6: Test
────────────
curl https://api.xoperr.dev/health
# Should return: {"status":"healthy","service":"Lessons API"}

🔧 TROUBLESHOOTING:
════════════════════

Service won't start?
  → Check logs: sudo journalctl -u lessons-api -n 50
  → Check API key is set in service file

502 Bad Gateway error?
  → Check if service is running: sudo systemctl status lessons-api
  → Check nginx error log: sudo tail -f /var/log/nginx/api.xoperr.dev.error.log

High memory usage?
  → If Whisper loaded (first STT call): Normal, it stays cached
  → To free memory: sudo systemctl restart lessons-api

SSL certificate issue?
  → Renew: sudo certbot renew --force-renewal
  → Check expiry: sudo certbot certificates
